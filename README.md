### Zicheng Wang

I build models and pipelines that read biology end to end, from raw reads to learned structure.

```
┌────────────────────────────────────────────────────┐
│  computational biology   ·   machine learning      │
│                                                    │
│  single cell → spatial → pathology → protein       │
│  sequence · structure · whole-slide image → model  │
└────────────────────────────────────────────────────┘
```

I work close to the bench, with the people who generate the data. Single-cell and
spatial genomics through QC, integration, and interpretation, and deep learning over
sequence, structure, and imaging at cluster scale.
